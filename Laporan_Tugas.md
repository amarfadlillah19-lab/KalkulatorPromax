## 2. Deskripsi Aplikasi
KalkulatorPromax adalah aplikasi web kalkulator canggih yang dibangun menggunakan bahasa Python dengan framework Flask untuk backend, serta HTML, CSS (Bootstrap 5), dan JavaScript untuk frontend. Aplikasi ini mendukung berbagai macam perhitungan, mulai dari aritmatika dasar, operasi logika, transformasi basis dan suhu, konversi mata uang statis, hingga perhitungan faktorial dan deret Fibonacci.

## 3. Fitur Utama yang Diimplementasikan
1. **Operasi Aritmatika:** Penjumlahan, pengurangan, perkalian, pembagian, pangkat, akar, modulus, dan floor division.
2. **Operator Logika:** AND, OR, NOT, XOR, NAND, NOR (mendukung input boolean maupun bitwise).
3. **Transformasi Bilangan:** 
   - Konversi Basis (Desimal, Biner, Oktal, Heksadesimal)
   - Konversi Suhu (Celcius, Fahrenheit, Kelvin, Reamur)
   - Konversi Mata Uang (IDR ke berbagai mata uang asing secara statis)
4. **Fitur Bonus:** Faktorial dan Deret Fibonacci.
5. **Kualitas UI/UX:**
   - Desain retro terminal modern yang responsif.
   - Dark/Light mode toggle.
   - Output yang menampilkan rumus, hasil, langkah detail perhitungan, dan log riwayat (history).
